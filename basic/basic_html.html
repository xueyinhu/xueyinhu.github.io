HTML (HyperText Markup Language)

超文本: 一种组织信息的方式，通过超链接将不同空间的文字、图片等各种信息组织在一起，能从当前阅读的内容，跳转到超链接指向的内容。>

标记：文本变成超文本需要用到的各种标记符号。

语言：每一个标记的写法、读音、使用规则，构成的标记语言。

注意事项

标签名不分大小写，但是推荐小写。

属性名、属性值都不区分大小写，但是推荐小写。

双引号，也可以写作单引号，甚至可以不写，但是推荐双引号。

标签中不要出现同名属性，否则后面写的无效。

代码中的多个空格、多个回车，都会被浏览器解析成一个空格。

虽然 a 标签作为行内元素，但 a 元素可以包裹除它自身外的任何元素。

HTML 结构

文档声明

HTML5 写法为 <!DOCTYPE html>。

必须写在网页的第一行，且在 html 标签的外侧。

字符编码

常见的字符集

ASCII：大写字母、小写字母、数字、一些符号， 共计128个。

ISO 8859-1：在 ASCII 基础上，扩充了一些希腊字符等， 共计256个。

GB2312：收录了 6763 个常用汉字与 682 个字符。

GBK：收录的汉字与符号达到 20000+，支持繁体中文。

UTF-8：包含世界上所有语言的所有文字与符号。

语言

写法为：<html lang="zh-CN">。

lang 属性的编写规则

语言-国家/地区，例如 zh-TW。

语言-具体种类，例如 zh-Hant ，已经不推荐使用。

标签

语义化

标签的默认效果不重要，用特定的标签表示特定的含义。

优势：代码结构清晰且可读性强；有利于 SEO(搜索引擎优化)；方便设备的解析(屏幕阅读器、盲人阅读器等)。

排版标签

<h1~h6> 标题

<p> 段落

<div> 无意义

文本标签

<em> 要着重阅读的内容。

<strong> 十分重要的内容。

<span> 没有语义，包裹短语。

<cite> 作品标题。

<dfn> 特殊术语或者专属名词。

<del> 删除的文本。

<ins> 插入的文本。

<sub> 下标文字。

<sup> 上标文字。

<code> 一段代码。

<samp> 在正常的上下文中，将某些内容提取出来。

<kbd> 键盘文本。

<abbr> 缩写，最好配合上 title 属性。

<bdo> 更改文本方向，要配合 dir 属性。

<var> 标记变量，可以与 <code> 一起使用。

<small> 附属细则，包括版本、法律文本。

<b> 摘要中的关键字，评论中的商品名称。

<i> 本意是人物的思想活动、所说的话等。现在多用于呈现字体图标。

<u> 与正常内容有反差文本，例如：错的单词、不合适的描述等。

<q> 短引用。

<blockquote> 长引用，块级元素。

<address> 地址信息，块级元素。

图像

图像标签

alt 属性的作用：搜索引擎得知图片的内容；有些浏览器在图片无法展示时展示 alt 的值；盲人阅读器会阅读 alt 的值。

图像格式
.jpg or .jpeg 格式

是一种有损的压缩格式。

支持的颜色丰富、占用空间较小、不支持透明背景、不支持动态图。

常用于网站的产品宣传图。

.png 格式

是一种无损的压缩格式。

支持的颜色丰富、占用空间较大、支持透明背景、不支持动态图。

常用于公司 logo 图或重要配图等。

.bmp 格式

不进行压缩，最大程度保留细节。

支持的颜色丰富、保留的细节更多、占用空间极大、不支持透明背景、不支持动态图。

大型游戏中的图片，网页中很少使用。

.gif 格式

仅仅支持 256 种颜色，色彩呈现不是很完整。

支持颜色较少、支持简单透明背景、支持动态图。

网页中的动态图像。

.webp 格式

谷歌推出的一种格式，专门在网页中呈现图片。

具备上述几种图像格式的优点，但是兼容性不太好，一旦使用就要解决兼容性问题。

可用于网页中的各种图片。

base64格式

要通过浏览器打开，传统看图应用无法打开。

将图像进行 base64 编码，形成一串文本。

适用于较小的图片，或者需要和网页同时加载的图片。

超链接

标签属性

target 属性决定跳转时如何打开页面，常用值如 _self(在本页签打开) 与 _blank(在新页签打开)。

跳转文件

若浏览器无法打开文件，则会引导用户进行下载。

若想强制触发下载，使用 download 属性。

跳转锚点

具有 href 属性的 a 标签是超链接，具有 name 属性的 a 标签是锚点。

name 与 id 都是区分大小写的，且 id 最好别是数字开头。

唤起指定应用

唤起设备拨号 herf="tel:10010"

唤起设备发送邮件 herf="mailto:10010@qq.com"

唤起设备发送短信 herf="sms:10086"
